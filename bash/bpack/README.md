BPACK
=========
bpack is a bash package manager.

The syntax is like pacman/PKGBUILD 

When the script runs. It sources the PKGBUILD (or whatever you call it.)
and then runs the install function.



Usage
-----

Quick example:

>> ./bpack install package name
